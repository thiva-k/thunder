# Express SDK

The <ProductName /> Express SDK (`@thunderid/express`) provides middleware and route handlers to integrate <ProductName /> authentication into Express applications.

## Installation

Install the <ProductName /> Express SDK using your preferred package manager:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">
    npm install @thunderid/express
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn add @thunderid/express
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm add @thunderid/express
  </CodeBlock>
</CodeGroup>

## Getting Started

To get started quickly, see the [Express Quickstart Guide](/docs/next/guides/getting-started/connect-your-application/express).

## Features

- Add <ProductName /> authentication middleware to your Express application
- Handle sign-in and sign-out with built-in route handlers
- Protect routes with middleware
- Access the authenticated user profile in route handlers
